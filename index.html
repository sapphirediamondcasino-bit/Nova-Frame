What is Nova Frame?

Nova Frame is a comprehensive security infrastructure platform developed by Kurrasch Systems Game Killer Studio. It provides enterprise-grade protection, storage, and API services for Discord bots, Roblox experiences, and gaming platform integrations.

Core Services

  • Secure Token Management: Encrypted storage and automatic rotation of authentication tokens
  • Partner Storage: Dedicated storage space for bot configurations and data
  • FDR (Failover Disaster Recovery): Automatic failover systems to ensure 24/7 uptime
  • Real-time Security Monitoring: Live threat detection and automatic blocking
  • Multi-Platform Support: Discord, Roblox, FiveM, LSPDFR, and more

How It Works

Partners integrate with Nova Frame through our secure REST API. Upon approval, you receive:

  • Unique authentication token for API access
  • Integration documentation and setup guides
  • Access to storage, security, and FDR services
  • Dashboard for monitoring and configuration

Important Note

Nova Frame provides infrastructure services - partners receive instructions and API access, NOT source code. All code remains protected by advanced security measures.

Security Model

Nova Frame employs multiple layers of security to protect partner data and prevent unauthorized access.

Authentication

  • Token-based Auth: Unique encrypted tokens for each partner
  • Auto-rotation: Tokens automatically rotate for enhanced security
  • Rate Limiting: Prevents brute-force and DDoS attacks
  • IP Whitelisting: Optional IP restriction for API access

Encryption

  • At Rest: AES-256 encryption for all stored data
  • In Transit: TLS 1.3 for all API communications
  • Token Storage: Tokens are hashed and salted

Port Security

Port: 25724 (Custom secured port) Protocol: HTTPS only Scanning Protection: Auto-block after 3 failed attempts Rate Limit: 100 requests/minute per token

Threat Detection

  • Port Scanning: Auto-block after 3 invalid connection attempts
  • Brute Force: Account lockout after 5 failed authentications
  • DDoS: Automatic traffic shaping and blackholing
  • Injection: Input sanitization at all entry points

Zero Tolerance Policy

Any attempt to breach, probe, or reverse engineer Nova Frame results in immediate termination, permanent IP ban, and potential legal action.

API Overview

Nova Frame provides a secure REST API for authenticated partners to interact with the platform. The same API key works across all supported platforms (Discord, Roblox, FiveM, etc.).

Base URL

https://api.novaframe.site:25724

Authentication

All API requests require an Authorization header:

Authorization: Bearer NF_[YOUR_TOKEN] Content-Type: application/json

Universal API Key

Your Nova Frame API key works across all platforms - Discord, Roblox, FiveM, and more. No need for separate keys per platform.

Endpoints

Method Endpoint Description
GET /api/status System status and metrics
GET /api/feed Live activity feed
POST /api/partner/apply Submit partnership application
GET /api/bots List authenticated user's bots (Discord & Roblox)
GET /api/bots/:id Get specific bot details
PUT /api/bots/:id/fdr Update FDR configuration
GET /api/roblox/verify Verify Roblox user authentication
POST /api/roblox/webhook Roblox event webhook receiver

Rate Limits

  • Standard: 100 requests/minute
  • Burst: 10 requests/second max
  • Daily: 10,000 requests

Error Responses

{ "error": true, "code": "RATE_LIMITED", "message": "Rate limit exceeded. Try again in 60 seconds.", "retry_after": 60 }

Privacy Policy

Data Collection

Nova Frame collects only essential operational data:

  • Bot authentication tokens (encrypted)
  • API request logs (timestamp, endpoint, client ID)
  • Security event logs (threats, blocks, attempts)
  • Storage operation metadata (not content)
  • Partnership application information
  • Discord user data for authenticated sessions (ID, username, email, guild list)
  • Roblox user IDs for verification (when applicable)

What We DO NOT Collect

  • Discord message content or chat history
  • Roblox in-game chat or player actions
  • Personal information beyond contact email
  • Bot internal operations or command usage
  • Server member data or activity logs
  • Payment information (handled by PayPal)

Data Usage

Collected data is used exclusively for:

  • System security and threat prevention
  • Bot authentication and authorization
  • Performance monitoring and optimization
  • Partnership application processing
  • Incident response and investigation

Data Security

  • AES-256 encryption for all sensitive data
  • Secure token storage with automatic rotation
  • Automated backup systems (every 100 writes)
  • Complete access logging and audit trails
  • Regular third-party security audits

Data Retention

Logs are retained for 90 days. Partnership data is kept indefinitely unless deletion is requested. Exiting partners may request full data deletion within 30 days of termination.
